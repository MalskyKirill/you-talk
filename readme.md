# Проект по курсу Основы frontend-разработки (YouTalk Layout) НИЯУ МИФИ

## Описание проекта

Статический фронтенд-проект с вёрсткой двух страниц сервиса YouTalk:

- `index.html` — страница блога.
- `article.html` — страница отдельной статьи.

## Структура проекта

```text
.
├── index.html
├── article.html
├── css/
│   ├── font.css
│   ├── style.css
├── fonts/
└── img/
```

## Локальный запуск

Открыть HTML-файлы в браузере:

1. `index.html`
2. `article.html`

Либо через Live Server в vscode.

## Стили

### Основной файл

- `css/style.css` — актуальные стили проекта, включая адаптивные брейкпоинты.


## Адаптив

В проекте используются брейкпоинты:

- `1200px`
- `1024px`
- `768px`
- `480px`

На этих ширинах изменяются:

- шапка (навигация/бургер),
- сетка карточек блога,
- лэйаут статьи и сайдбара,
- футер,
- размер и поведение отдельных UI-элементов.

## Примечание

Проект не использует сборщик, фреймворк и npm-зависимости — только HTML/CSS и статические ассеты.
